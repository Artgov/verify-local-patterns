Parking start page

What we learnt

None of the users we have spoken to have had a problem with checking to see if they needed a permit before they started.

Early versions of this page did not mention the types of permits that would be available and this led to uncertainty for some. These users were unsure what permit the service would let them apply for.

What we changed

To address users' uncertainty we added content to let users know which permits they could apply for with this service.
