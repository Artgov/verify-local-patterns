Linked to from the service start page, this page should communicate:

  • that users can prove their identity online using GOV.UK Verify, and can create an account
  • alternative online and offline options for proving identity
  • how long the process of applying will take
  • when they should expect to receive their bus pass once they've applied
See the example council page

GOV.UK Verify

If the user selects the GOV.UK Verify route, they will be sent away from the council's website to prove their identity to a certified company. Before the user does this, the council should make sure the user understands what personal data will be shared so that the user can change their mind with minimal wasted time.
