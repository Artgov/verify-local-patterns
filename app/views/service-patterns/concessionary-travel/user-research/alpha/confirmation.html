Concessionary travel start page

What we learnt

At the end of the transaction, users need a clear confirmation that they have completed the task. In addition to this, they also need to understand when and how they will recieve their pass.

Several users also wanted to know how and where they would be able to user their pass.

What we changed

Content has been added to this page to explain how the pass can be used.

Content has also been added to mention other potentially relevant services the user can access with their Verify account. This addresses an observed issue where some users thought the identity verification process was specifically for the bus pass application.
